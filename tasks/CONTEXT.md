## Workflow

1. Start session → read `todo.md` + `lessons.md`
2. Pick next `[ ]` item
3. Implement → verify → mark `[x]`
4. After any correction → update `lessons.md` with the pattern
5. At sprint end → archive sprint in `todo.md`, propose next sprint
